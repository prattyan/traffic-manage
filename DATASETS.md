# Public Datasets

This document provides links and download instructions for various public datasets used in traffic management research.

## BDD100K
- **Description**: The Berkeley DeepDrive (BDD) dataset is a large-scale driving video dataset for various autonomous driving tasks.
- **Link**: [BDD100K Dataset](https://bdd100k.com/)
- **Download Instructions**:
  1. Visit the link above.
  2. Follow the instructions on the site to request access.
  3. Download the dataset using the provided scripts or manual download options.

## UA-DETRAC
- **Description**: The UA-DETRAC dataset is designed for vehicle detection and tracking tasks. It provides a diverse set of videos with varying traffic conditions.
- **Link**: [UA-DETRAC Dataset](http://detrac-db.rit.albany.edu/)
- **Download Instructions**:
  1. Navigate to the page using the link.
  2. Follow the instructions to register and gain access.
  3. Download the videos using the server or provided links.

## METR-LA/PEMS-BAY
- **Description**: METR-LA and PEMS-BAY datasets are used for traffic forecasting based on historical traffic data.
- **Link**: [METR-LA Dataset](https://github.com/google-research/google-research/tree/master/traffic_forecasting)
- **Download Instructions**:
  1. Visit the GitHub page.
  2. Follow the setup instructions for downloading and preprocessing the data.
  3. Ensure to adhere to any usage guidelines provided.
